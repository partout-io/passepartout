Защитите свою конфиденциальность в интернете с Passepartout, умным VPN-клиентом, который идеально интегрирован с iOS, macOS, iPadOS и tvOS.

КАК РАБОТАЕТ PASSEPARTOUT

Passepartout включает базовые функции импорта и подключения. Дополнительные функции требуют покупки в приложении.

ВКЛЮЧЕННЫЕ ФУНКЦИИ:

- Поддержка протоколов OpenVPN и WireGuard.
- Скрытие OpenVPN (с помощью --scramble).
- Автоматизация с помощью команд Siri для упрощения работы.
- Исключение конечных точек для точного контроля.

ТРЕБУЕТ ПОКУПКУ В ПРИЛОЖЕНИИ:

- Поддержка Apple TV.
- Автоматические правила подключения для умных и безопасных соединений.
- Синхронизация через iCloud для использования на нескольких устройствах.
- DoH/DoT для безопасных DNS-запросов.
- Сопоставление доменов для split DNS.
- Экспорт и совместное использование профилей.
- Индивидуальная маршрутизация для продвинутых настроек.
- Интерактивный вход с поддержкой одноразовых паролей (OTP).
- Профили без VPN (например, настройки только DNS/прокси).

СЕМЕЙНЫЙ ДОСТУП

Все покупки поддерживают функцию «Семейный доступ», поэтому вы можете делиться ими с членами вашей семьи.

КОНФИДЕНЦИАЛЬНОСТЬ, КОТОРОЙ МОЖНО ДОВЕРЯТЬ

Passepartout разработан с учетом прозрачности и конфиденциальности:  
- Узнайте, к каким серверам вы подключаетесь.  
- Без рекламы, трекеров и ненужных фоновых процессов — только необходимое VPN-соединение.
